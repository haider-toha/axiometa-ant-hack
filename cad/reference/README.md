# Reference: Axiometa Genesis Mini shell

Known-good 3D-printed shell for the **Axiometa Genesis Mini** (MTX0013). Someone else designed and printed this enclosure; it actually fits the board and modules in real life, so it is the fit reference for our own CAD work.

## Source

[Axiometa Genesis Mini Game Console Shell](https://www.printables.com/model/1767025-axiometa-genesis-mini-game-console-shell) by Nebenezer on Printables (for Axie's Arcade / starter-kit modules).

## Files in this folder

| File | What it is |
|------|------------|
| `genesis-mini-shell.step` | STEP of the reference shell geometry |
| `image_of_the_build.png` | Photo of a physical print of this shell (grey body, red encoder knob / accents) |

## Module layout (as designed for this shell)

| Port | Module |
|------|--------|
| P1 | AX22-0003 — Rotary encoder |
| P2 | AX22-0050 — LED button |
| P3 | AX22-0034 — 0.96" IPS LCD (80×160) |
| P4 | AX22-0018 — Passive buzzer |

Also exposes the board-mounted button (#45) and USB-C. Original print notes: PLA+, optional encoder knob cover, (4)× 2×20 mm screws.

## Why keep this here

Use this STEP (and the photo) when checking our enclosure CAD against a shell that is already proven to seat the Genesis Mini and its modules — openings, clearances, and overall envelope — before trusting our own models alone.
