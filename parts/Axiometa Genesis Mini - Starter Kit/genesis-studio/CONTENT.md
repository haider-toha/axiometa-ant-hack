## Description

Axiometa Studio (also called Genesis Studio) is the prompt-to-firmware web IDE included with the Genesis Mini Starter Kit. Describe a project in natural language and it generates Arduino-oriented firmware (and optional dashboard UI) for the ESP32-S3 Genesis Mini and AX22 modules. Site tagline: "From prompt to device." It pairs modular AX22 hardware (sensors, LEDs, motors — no soldering) with firmware generation. Not a hardware SKU; storefront kit "Learn More" for Studio points here.
